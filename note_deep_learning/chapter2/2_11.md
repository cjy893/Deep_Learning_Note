# 向量化
## 什么是向量化
以logistic回归中,$z=w^T\vec{x}+b$为例

$w,x$都是n维列向量,所以在计算中,如果使用了非向量化的方法,以**python**为例,就会有
```
z = 0
for i in range(n)
    z += w[i]*x[i]

z +=b
```
这样的计算效率较低，而向量化的计算就非常直接\
**你可以使用如下命令**
```
import numpy as np
z = np.dot(w,x) +b
```
经过检验，可以发现向量化的计算速度快于循环体遍历，且消耗时间是循环体的接近三百分之一
****
所以得出经验，***只要有其他可能，最好不要使用显式的for循环***
