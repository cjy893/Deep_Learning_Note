# 向量化logistic回归

## 正向传播

**首先,我们需要进行logistic回归的正向传播步骤**
对第一个样本进行预测,运用公式,计算出该样本的$z,\hat{y}$,以此类推,直到将所有样本计算完成

在这个过程中,我们甚至也可以将**for循环**简化

---

首先给出训练集矩阵$X_{n×m}$以及参数矩阵$W$
构建行向量$\vec{z},\vec{b}$

```
z = np.dot(W,X) + b
```

然后构建向量$\vec{a}$

```
a = 1/(np.exp(-z) + 1)
```

所以在对logistic回归的正向传播中，我们也将计算进行了向量化，可以通过这一步骤，省略**for循环**，同时计算所有数据
